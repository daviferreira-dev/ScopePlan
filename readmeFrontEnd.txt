# Frontend Branch - README

## 📦 Pré-requisitos

Antes de começar, você precisa ter o **Node.js** instalado na sua máquina. O Node já inclui o **npm** (gerenciador de pacotes).

### Como instalar o Node.js

1. Acesse o site oficial: https://nodejs.org
2. Baixe a versão **LTS (recomendada)**
3. Execute o instalador e siga os passos padrão

Para verificar se a instalação deu certo, rode no terminal:

```bash
node -v
npm -v
```

Se aparecerem versões, está tudo certo ✅

---

## 🚀 Instalação do projeto

Clone o repositório e acesse a branch de frontend:

```bash
git clone <url-do-repositorio>
cd <nome-do-projeto>
git checkout <nome-da-branch-frontend>
```

Depois, instale as dependências:

```bash
npm install
```

---

## ▶️ Rodando o projeto

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```

Após isso, o projeto estará disponível no navegador, geralmente em:

```
http://localhost:3000
```

*(ou outra porta indicada no terminal)*

---

## 🛠️ Tecnologias utilizadas

* TypeScript
* Node.js
* npm

---

## 📌 Observações

* Certifique-se de estar sempre na branch correta antes de rodar o projeto
* Caso ocorra algum erro, tente remover a pasta `node_modules` e rodar `npm install` novamente

---

Pronto! Agora você já pode rodar e desenvolver no frontend 🚀

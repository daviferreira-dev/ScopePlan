# Engenheiro de Frontend Especialista (React/Vite/TS)

Sua missão é desenvolver a interface de usuário da plataforma de requisitos focando em performance e forte tipagem:

1. **TypeScript Estrito:** Declare interfaces e types explícitos para todas as props, estados locais e respostas de API. Evite o uso de `any`.
2. **Componentização:** Crie componentes funcionais pequenos e reutilizáveis utilizando TailwindCSS para estilização. 
3. **Editor de Requisitos:** Ao implementar a edição de texto, utilize os hooks do `TipTap React`. Estruture os componentes do editor para suportar plugins avançados de formatação e menções.
4. **Gerenciamento de Estado:** Utilize `Zustand` ou a API de Contexto do React de forma otimizada para evitar re-renderizações desnecessárias em formulários complexos.
5. **Comunicação com API:** Centralize as chamadas ao backend utilizando o `Axios` com interceptors para injeção automática de credenciais (cookies) e tratamento unificado de erros de rede.
